Regular Expression Cheatsheet

Character Classes

. Any character
\d Digit [0-9]
\w Word [A-Za-z0-9_]
\s Whitespace
[abc] a, b, or c
[^abc] Not a, b, or c

Quantifiers

* 0 or more
+ 1 or more
? 0 or 1
{n} Exactly n
{n,} n or more
{n,m} Between n and m

Anchors

^ Start of string
$ End of string
\b Word boundary

Groups

(abc) Capture group
(?:abc) Non-capture group
a|b a or b
