## 신규 기능 요약

### 13. 전생 테스트 ⭐⭐⭐
- 이름 + 생년월일 + 성별 입력
- 4개 시대 (고대/중세/아시아/신비) × 다양한 직업
- 전생 이야기, 성격, 카르마, 조언 제공
- 공유하기 좋은 결과 카드

### 14. 동물 성격 테스트 ⭐⭐⭐
- 8가지 성격 질문
- 8종류 동물 (사자/고양이/강아지/여우/부엉이/늑대/나비/토끼)
- 성격 특성, 장단점, 궁합 동물
- 귀여운 동물 이모지 결과

### 15. AI 관상 분석 ⭐⭐⭐
- 5단계 선택 (이마/눈/코/입/얼굴형)
- 4가지 운세 점수 (연애/재물/직업/건강)
- 길상/평상 등급 판정
- 상세 분석 텍스트

### 16. 연애 성향 테스트 ⭐⭐⭐
- 10개 연애 상황 질문
- 7가지 연애 유형 (올인형/안정형/자유형/로맨틱/썸타는/헌신형/주도형)
- 성향 퍼센트 그래프
- 궁합 유형, 조언

### 17. 전생의 인연 ⭐⭐
- 두 사람 이름 + 생년월일 입력
- 8가지 전생 관계 스토리
- 카르마 분석 (신뢰/열정/운명/조화)
- 커플용 공유 최적화

### 18. 2025 한마디 ⭐⭐⭐
- 이름 + 생년월일 + 희망사항 입력
- 새해 메시지 + 3가지 키워드
- 행운 요소 (색/방향/숫자/요일)
- 월별 운세 표시
- 시즌 트래픽 폭발 기대
